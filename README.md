# Salary Prediction Using Polynomial Regression

## Project Overview
This project predicts employee salary based on years of experience using Polynomial Regression, a type of Non-Linear Regression.

## Dataset
- Salary_Data.csv
- Input: YearsExperience
- Output: Salary

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Steps Performed
1. Loaded the dataset from CSV file.
2. Selected YearsExperience as input (X).
3. Selected Salary as output (Y).
4. Converted data using Polynomial Features.
5. Trained the model using Polynomial Regression.
6. Predicted salary values.
7. Visualized the results using a graph.

## Why Polynomial Regression?
Linear Regression fits a straight line.

Polynomial Regression can fit a curve and capture more complex relationships between experience and salary.

## Output
The model predicts salary based on years of experience and displays the prediction curve on a graph.
